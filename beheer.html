GitHub Deploy Sync

Deze beheeromgeving is GitHub-first: zonder token zijn bewerkingen uitgeschakeld. Wijzigingen worden direct naar GitHub geschreven (events.json, standpunten.json, bestuur.json), zodat deploy dit automatisch meeneemt.

GitHub sync is verplicht om te kunnen bewerken. Vul owner/repo/token in en klik op “Config opslaan”.
